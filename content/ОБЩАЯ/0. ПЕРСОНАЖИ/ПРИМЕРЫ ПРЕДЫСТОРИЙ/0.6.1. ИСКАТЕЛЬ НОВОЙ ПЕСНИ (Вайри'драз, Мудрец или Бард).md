
Вы провели десятилетия на вершине мира, слушая, как Песнь Древа превращается из симфонии в похоронный марш. Ваш старейшина сказал: «Тот, кто слышит болезнь, должен найти лекарство». Вы спустились с гор не для приключений, а с **миссией**: найти источник угасания Песни или, если её не спасти, **записать Новую Песню**, которая сможет родить следующий мир, когда этот падёт. Вы носите с собой свиток чистейшего пергамента и резец из горного хрусталя — инструменты для записи последней великой мелодии.

- **Личная цель:** Записать ключевое событие (подвиг, жертву, открытие), которое, по вашему мнению, может стать «семенем» для новой реальности.
